3.2. Design Pattern

Design Patterns (Entwurfsmuster) sind (Muster-)Lösungen für wiederkehrende Probleme beim Entwurf von Software. Sie erleichtern die Entwicklung und Architektur von großen Softwareprojekten.

Auflistung einiger Design Patterns

  • Class Adapter
  • Composite
  • Decorator
  • Factory
  • Iterator
  • Object Adapter
  • Observer
  • Singleton
  • State
  • Strategy
  • Template Method
